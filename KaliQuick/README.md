# KaliQuick
Bash script for downloading some tools I like to have upon loading a new Kali installation. 

## Prerequisites
- Installation of go

## Tools Included
- Sublime Text
- pip3, pip
- python setuptools
- Sublist3r
- Subfinder
- Github Search
- Amass
- DirSearch
- SubDomainizer
- gau (get all urls)
- GoSpider
- EyeWitness
- Masscan
- MassDNS
